## 소프트웨어 정의 및 제공 기능
* 1-3차년도에 개발된 모니터링 기능 및 복원 기능을 하나로 통합한 구조 설계 및 구현
* 기능 검증을 위해 IoT 클라우드 환경에 적용하여 기능 테스트를 실시함
* 템플릿을 이용하여 각 서비스 특성에 따른 장애 탐지 및 복원 알고리즘 적용 가능
